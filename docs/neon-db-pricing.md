# Neon Database Pricing

**Source:** https://neon.com/pricing

**Typical spend:** $15 /mo  
**Based on:** Intermittent load, 1 GB

All features in the Free tier, plus:

- **Neon Auth:** 1M MAUs
- **Time Travel / Restores:** 7-day window
- **Metrics / Logs in UI:** 3-day retention
